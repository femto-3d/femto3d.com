Project Overview

The BCI-Controlled Bionic Arm is a fully functional prosthetic arm built entirely from first principles — no commercial prosthetic kits used. Every component was designed, fabricated, and programmed independently: the mechanical structure was modelled in Fusion 360 and 3D-printed; the control electronics used a custom-designed PCB with ARM Cortex-M microcontroller; and the software stack combined real-time EEG-based brain-computer interfacing with a Python/TensorFlow computer vision safety gate.

The project introduced a novel dual-input safety architecture — the arm actuates only when both conditions are simultaneously satisfied: the user's EEG attention level exceeds a calibrated threshold AND the TensorFlow object detection pipeline confirms the target object is present in the camera frame. This prevents any unintended limb movement and makes the system genuinely safe for clinical use.

The project won the TNSI (Tamil Nadu Startup & Innovation) 2022 National Final Round — awarded INR 1,00,000 by the Government of Tamil Nadu — and was also a Runner-Up at the IEEE Region 10 (Asia-Pacific) Undergraduate Project Video Contest 2021.

How It Was Built — Step by Step

1

Mechanical Design — Autodesk Fusion 360

Designed the full prosthetic arm structure in Fusion 360 — anatomically contoured, lightweight, and modular. Each finger segment was designed to house a servo motor horn with clearance for tendons. Multiple design iterations were 3D-printed in PLA, tested for range of motion, and refined. Key constraints: weight <400g, full grip force >5N, all standard servo mounts compatible.

2

Custom PCB — Dual-Stage Bio-Potential Amplifier (Autodesk Eagle + TL074)

Designed a 2-layer PCB in Autodesk Eagle for conditioning EMG/EEG bio-potential signals (µV–mV range) before feeding them to the ARM Cortex ADC. Stage 1: differential instrumentation amplifier with high CMRR for EMG surface electrode rejection. Stage 2: active bandpass filter (20–500 Hz for EMG; 8–30 Hz for EEG alpha/beta) using the TL074 quad op-amp. Validated on oscilloscope — clean output with <2% noise after conditioning. → Full PCB project detail

3

Firmware — Embedded C on STM32 / ARM Cortex-M

Programmed multi-finger servo motor actuation firmware in Embedded C on STM32 (ARM Cortex-M3). Used timer-based PWM generation for 5 independent servo channels — each finger controlled individually. Implemented smooth interpolation between grip positions using a lookup table for 8 predefined gestures (open, close, pinch, lateral grip, etc.). Serial UART interface receives commands from the Python control layer.

4

EEG BCI Integration — Brain-Computer Interface

Integrated a commercial EEG BCI headset module that outputs a real-time "attention score" (0–100) via serial/Bluetooth. The Python script reads this score at 10 Hz and applies a sliding-window average to reduce noise. When the smoothed attention value exceeds the user-calibrated threshold (typically 65–75/100), the intent flag is raised. This threshold is user-tunable to accommodate different cognitive baselines.

5

Computer Vision — TensorFlow + OpenCV Object Detection

Built a real-time object detection pipeline using TensorFlow (SSD MobileNet) and OpenCV running on a Raspberry Pi. The camera feed is processed at 15 fps — detecting and classifying target objects (cups, bottles, tools). When a recognized object appears in the "grab zone" (lower 30% of frame), the vision gate is set. This acts as the second independent safety condition. Object class whitelist is configurable.

6

Dual-Input Safety Gate & Integration

The Python orchestration layer runs on Raspberry Pi and merges both inputs: if eeg_attention_flag AND vision_object_flag: send_grip_command(). Only when BOTH conditions are simultaneously true does a grip command get sent over serial to the STM32. This eliminates accidental actuation from EEG noise alone or from the vision system seeing an object the user isn't focused on. False positive rate in testing: <1.5%.

7

Testing, Iteration & Award

Conducted 200+ live grasp trials. Grip success rate: >91%. Demonstrated at TNSI state and national rounds with live amputee volunteers. Judges specifically praised the dual-input safety gate as "a first-of-its-kind safety architecture for student prosthetics." Won the TNSI 2022 National Final — INR 1,00,000 — and was Runner-Up at IEEE Region 10 Asia-Pacific contest.
